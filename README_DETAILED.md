# Voxa - Voice AI Workspace Agent

## Description
A mobile-first voice AI workspace agent for Google Workspace and Microsoft 365, designed to replace typing with intelligent voice commands.

## Core Tech Stack
Expo (React Native), FastAPI, DeepAgents, Composio, Groq Whisper, Gemini TTS.

## Key Features
- Wake-word detection and voice capture via Expo
- Real-time transcription via Groq Whisper
- AI action planning via DeepAgents
- Secure API integration with Google Workspace via Composio
- Human-in-the-loop approval sheets for destructive actions

## Major Learnings
Building mobile voice-to-text pipelines, handling audio in React Native, and designing safe agentic workflows with explicit user approval gates.
